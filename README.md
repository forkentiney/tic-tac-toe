# Tic Tac Toe
A Tic Tac Toe game created as part of The Odin Project Curriculum

I'm aware that there is a bug on this site whereby after a player has won, consecutive moves can be made which normally add points to player 1's score. This happens because there is no check for when the game is over, but I am not going to add it at this time. Further, points are only normally added to player 1's score (even if player 2 won the game) because of the way scores are calculated and checked. Adding the aforementioned check would fix both of these bugs.
